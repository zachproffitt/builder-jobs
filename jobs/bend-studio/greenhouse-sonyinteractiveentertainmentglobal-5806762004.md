Are you looking for a challenge?&nbsp;

We are seeking a Staff Software Engineer specializing in React Native and high-performance frontend development for mobile and desktop platforms. In this role, you will define the architecture, frameworks, and technical best practices that drive PlayStation’s mobile and desktop user experiences.&nbsp;

&nbsp;

You will collaborate closely with native macOS engineers, product, and design teams to deliver best-in-class user interfaces that are performant, accessible, and delightful at scale. As a Staff Engineer, you will provide technical leadership across teams, mentor engineers, and push the boundaries of what’s possible in cross-platform frontend development.&nbsp;

&nbsp;

Responsibilities&nbsp;

Define and evolve the frontend architecture for PlayStation client applications on mobile using React Native and modern JavaScript/TypeScript.&nbsp;

Drive performance optimizations and best practices for rendering, startup time, and UI responsiveness.&nbsp;

Collaborate with native platform engineers (C++/Objective-C/Swift) to design and integrate custom native modules and optimize cross-platform interoperability.&nbsp;

Apply architectural design patterns such as MVC, MVVM, MVP to develop a sustainable and reusable codebase.&nbsp;

Partner with product managers, designers, and researchers to translate user experience requirements into elegant, maintainable solutions.&nbsp;

Establish coding standards, testing strategies, and development patterns that raise the quality bar for frontend engineering.&nbsp;

Mentor engineers and act as a multiplier for frontend development expertise across the broader team.&nbsp;

Investigate new frontend technologies, frameworks, and performance techniques, and guide their adoption when appropriate.&nbsp;

Represent frontend/OS needs in technical and architectural discussions across Sony Interactive Entertainment.

&nbsp;

&nbsp;Required Qualifications&nbsp;

8+ years of software development experience, with significant experience building React Native apps or high-performance JavaScript/TypeScript applications.&nbsp;

BS/MS degree in Computer Science or equivalent.&nbsp;

Strong expertise in frontend performance profiling, optimization, and debugging.&nbsp;

Passion for delivering world-class user interfaces with exceptional performance.&nbsp;

Deep understanding of cross-platform architecture and how to bridge between JS and native modules.&nbsp;

Proven track record of delivering production-quality mobile or desktop applications with React Native.&nbsp;

Experience designing scalable frontend architectures and reusable UI frameworks.&nbsp;

Strong collaboration skills and ability to work closely with native engineers, designers, and product managers.&nbsp;

Demonstrated technical leadership: leading projects, mentoring engineers, or setting engineering strategy.&nbsp;

&nbsp;

Desired Qualifications&nbsp;

Familiarity with iOS development and native bridging concepts (C++, Objective-C, Swift).&nbsp;

Experience with accessibility, internationalization, and UI testing frameworks.&nbsp;

Contributions to open-source React Native or frontend libraries.&nbsp;

Familiarity with CI/CD pipelines and modern build systems for frontend development.&nbsp;

Experience working with globally distributed teams.&nbsp;
