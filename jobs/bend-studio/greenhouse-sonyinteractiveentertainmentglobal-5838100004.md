Software Development Engineer in Test - Contractor
San Mateo, CA
Do you have a passion for test automation? We are looking for self-motivated, quick-starter and highly proactive individuals to join our full stack team, who will be testing new and existing features on multiple platforms and devices in a software development environment and ensuring quality of automation and test frameworks.
At PlayStation, we are passionate about both collaboration and technology. We love building the systems that millions of people use and are our own first customers. We strive to be at the forefront of development technologies and are a diverse company with teams on four continents.
Primary Responsibilities:
Plan and complete QE related activities; build test plan, test cases, and implement to ensure the quality of new features
Review product requirements and technical design documents to understand how the feature works on the PlayStation console and create effective testing strategies
Build, maintain and improve test automation frameworks, and help teams set up automation infrastructure
Develop automated UI and API tests to increase automation coverage
Report, track and verify defects as well as monitor automation jobs. Provide status reports to leadership and other key partners.
Collaborate with distributed teams for multi-functional test plan design and execution
Requirements:
BS or MS in Engineering, Computer Science or related field, or equivalent
3-5 years or equivalent experience in UI automation for web applications
Solid experience writing automated tests in Python, JavaScript, C ++ or any other Programming Language
Experience with Selenium or Appium or UI Automation technologies
Knowledge and experience testing APIs (RESTful), Libraries, SDK/Application Frameworks, Native Applications
Experience with at least one modern web application framework, preferably React
Strong technical, analytical and problem-solving skills
Able to work efficiently as part of a team and independently
Strong initiative and outstanding verbal and written communication skills
Bonus Qualifications:
Experience developing or writing test code/scripting for a game console or set-top box
Experience developing an application (game application is extra bonus)
Practical experience using development and deployment tools: Git, Jenkins, Docker
Experience using source control (esp. Git) and CI/CD tools (Jenkins or Circle CI)
