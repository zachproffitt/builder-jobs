At Rockstar Games, we create world-class entertainment experiences.&nbsp;

Become part of a team working on some of the most rewarding, large-scale creative projects to be found in any entertainment medium - all within an inclusive, highly-motivated environment where you can learn and collaborate with some of the most talented people in the industry.

Rockstar is on the lookout for a passionate Software Engineer who considers software development to be a craft, and&nbsp;possess a passion for solving complex problems at scale.&nbsp;&nbsp;

This is a full-time, in-office position based out of Rockstar’s new state-of-the-art game development studio in Carlsbad, CA.

WHAT WE DO

The Rockstar Games Online Services team creates connected experiences that are found across our game franchises.&nbsp;

We create online services for our games, including matchmaking, cloud saves, leaderboards, player inventory, player stats,&nbsp;bounties&nbsp;and more.&nbsp;

We improve existing features as well as architect&nbsp;new solutions&nbsp;to support the development of the Online Services features.&nbsp;

RESPONSIBILITIES

Develop high volume, highly scalable server-side features using Windows Server, SQL server, IIS, .NET, and C#.&nbsp;

Develop web services supporting game console clients, browser clients, and external server clients.&nbsp;

Develop&nbsp;highly reliable&nbsp;distributed server systems.&nbsp;

Web performance optimization.&nbsp;

Collaboration with other Rockstar technology teams across our worldwide studios.&nbsp;

REQUIREMENTS

4+ years of experience developing scalable distributed systems.&nbsp;

Expert in C# and/or C++.&nbsp;

Strong CS fundamentals, including excellent understanding of OO design patterns and concepts.&nbsp;&nbsp;

Experience with API design and development.&nbsp;

Knowledge of web service scalability, performance, and security practices.&nbsp;

Database development experience (MSSQL).&nbsp;

Experience with source control, preferably Perforce.&nbsp;&nbsp;

PLUSES

Please note that these are desirable skills and are not required to apply for the position.

Experience building online gaming systems, commerce systems, social networking systems, search&nbsp;systems.&nbsp;

One or more titles shipped on console or PC; preferably on the Xbox and/or PlayStation consoles.&nbsp;

Familiarity with networking systems and their components including DNS, NAT, firewalls, security methods, peer-to-peer&nbsp;and client server models.&nbsp;

Experience with online game service APIs, particularly Xbox Live, Sony NP, and Games for Windows Live.&nbsp;

Knowledge of Agile software development processes.&nbsp;
